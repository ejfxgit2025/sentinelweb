Core Features

Everything you need to maintain absolute focus.

AI Page Analysis

Analyzes page content in real-time to determine if it aligns with your current goal.

Strict Mode

Instantly blocks distracting sites and redirects you back to your work environment.

Warning Overlay

Pulsing red borders and visual alerts when you start drifting off-task.

Time Tracking

Precise monitoring of your Focus Time vs. Wasted Time throughout the day.

Score System

Gamified focus scoring that updates dynamically based on your browsing behavior.

Voice Assistant

Integrated ElevenLabs voice synthesis to verbally warn you when distracted.

How It Works

A seamless flow from intention to execution.

1. Set Your Goal

Tell Sentinel what you're working on (e.g., "Coding a React app").

2. AI Analysis

Sentinel silently analyzes every tab and page you visit.

3. Warning Triggered

If you open Reddit, the screen flashes red and the AI warns you.

4. Strict Redirect

If you ignore the warning, Sentinel closes the tab and redirects you.

5. Score Updated

Your focus score adjusts, keeping you accountable all day.

Install Sentinel

Get started in less than 2 minutes.

1

Download the Extension

Get the latest ZIP file from our repository.

2

Open Chrome Extensions

Navigate to chrome://extensions/ in your browser.

3

Enable Developer Mode

Toggle the switch in the top right corner.

4

Load Unpacked

Click "Load unpacked" and select the Sentinel folder.
